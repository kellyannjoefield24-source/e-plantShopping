# Paradise Nursery Portfolio Project

## Project Overview
An interactive React-based web application for a plant nursery shop. This project demonstrates state management using Redux Toolkit, responsive UI design, and dynamic shopping cart functionality.

## Core Features
*   **Landing Page:** Featuring the "Paradise Nursery" brand, a high-quality background image, and a "Get Started" call-to-action.
*   **Product Listing:** A dynamic grid of various plant categories (Air Purifying, Aromatic, etc.) with the ability to add items to a cart.
*   **Shopping Cart:** A full-featured cart that calculates subtotals and totals, allowing users to increment, decrement, or remove items.
*   **Navigation:** A responsive header with a live-updating cart icon showing the total number of items selected.
*   **Confirmations:** Interactive pop-ups to confirm successful recommendation submissions.

## Technical Details
*   **Frontend:** React.js
*   **State Management:** Redux Toolkit (Slices for Cart logic)
*   **Styling:** CSS3 with custom background implementations
*   **Deployment:** GitHub Pages / Public Repository
